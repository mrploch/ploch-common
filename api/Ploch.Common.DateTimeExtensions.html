ToEpochSeconds(DateTime)

Converts a DateTime to Epoch Seconds (Unix Timestamp - seconds since 00:00:00 UTC on 1 January 1970).

Parameters
Type Name Description
DateTime dateTime

The date time.
Returns
Type Description
long

Epoch seconds value.
Remarks

Helper method which simply casts DateTime to DateTimeOffset and calls ToUnixTimeSeconds().

This class doesn't do any manipulations of time-zones, so the time zone of the provided instance of DateTime will be used.
