ToEpochSeconds(DateTime)

Converts a System.DateTime to Epoch Seconds (Unix Timestamp - seconds since 00:00:00 UTC on 1 January 1970).

Parameters

System.DateTime dateTime

Type Description
System.Int64

Epoch seconds value.
Remarks

Helper method which simply casts DateTime to System.DateTimeOffset and calls System.DateTimeOffset.ToUnixTimeSeconds.

This class doesn't do any manipulations of time-zones, so the time zone of the provided instance of DateTime will be used.
