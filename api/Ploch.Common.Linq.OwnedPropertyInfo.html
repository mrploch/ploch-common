Class OwnedPropertyInfo

Represents an interface for retrieving and setting the value of a property owned by an instance of an object.

public abstract class OwnedPropertyInfo : IOwnedPropertyInfo
Remarks

This interface represents a property in a specific instance of an object. Instances of this interface provide an easy way to retrieve and set the value of a property. It wraps the PropertyInfo and connects it with a specific owning object instance.

It is extensively used by the ExpressionExtensions class.
